# Card-Based City Builder
Game Design & Technical Overview

## 1. High-Level Concept

A **2D city-building strategy game where buildings are represented as cards**.

Players draw cards each turn and place them on a **grid-based city map**. Cards interact with nearby cards through **simple adjacency rules and synergies**.

Core gameplay focuses on:
- spatial planning
- resource balancing
- card synergies
- incremental city growth

Typical run length: **20-40 minutes**.

---

## 2. Core Gameplay Loop

1. Draw cards
2. Place cards on the city grid
3. Resolve card effects
4. Update resources
5. Check events
6. Next turn

Loop:

draw -> place -> resolve -> update -> next turn

## 4. Card System

Each card contains:

- cost
- category
- base effects
- adjacency effects

Example:

Apartment Block  
Cost: 3

Base Effects  
+3 population  
+1 gold

Adjacency  
+2 happiness if next to Park  
-2 happiness if next to Factory

---

## 5. Card Categories

### Residential
- generates population

### Industry
- generates money
- creates pollution

### Services
- boosts nearby housing

### Infrastructure
- enables city growth

### Culture
- increases happiness

---

## 6. Adjacency System

Cards interact with neighbors:

- up
- down
- left
- right

Example rules:

Park next to Housing -> +2 happiness  
Factory next to Housing -> -3 happiness  
School next to Housing -> +1 population  
Market next to Housing -> +2 gold

---

## 7. Resources

### Gold
- used to place cards

### Population
- generated by housing

### Happiness
- citizen satisfaction

### Pollution
- produced by industry

---

## 8. Turn Structure

1. Draw phase
2. Placement phase
3. Resolution phase
4. Resource update
5. End phase

---

## 9. Victory Condition

Reach target population.

Example:

Population >= 1000

---

## 10. Loss Conditions

### Bankruptcy
Gold < 0

### Civil unrest
Happiness <= 0

### Population collapse
Population drops below minimum viable threshold.

---

## 11. Run Structure

Each play session is a **run** with randomized cards and an evolving city.

Run length: **20-40 minutes**.
